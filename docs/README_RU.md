# Документация RaiderEngine (RU)

Этот набор документов рассчитан на 2 сценария:

- расширение движка (архитектура, новые модули, backend-ы, editor-панели);
- разработка игры на существующей базе (ECS, системы, скрипты, уровни, сборка).

## С чего начать

1. Быстрый запуск: `docs/GETTING_STARTED_RU.md`
2. Архитектура: `docs/ARCHITECTURE_RU.md`
3. Расширение движка: `docs/ENGINE_EXTENSIBILITY_RU.md`
4. Работа с редактором: `docs/EDITOR_GUIDE_RU.md`
5. Разработка игры: `docs/GAME_DEVELOPMENT_RU.md`

## Карта тем

- Архитектура runtime и порядок обновления систем.
- ECS-подход и добавление новых компонентов/систем.
- Рендер backend-ы (`DirectX12`, `Vulkan`, `Null`) и путь расширения.
- Система скриптов на C# (`ScriptRuntime`) и интеграция в C++.
- Редактор: меню, панели, docking-слоты, сериализация уровня.
- Практический workflow от идеи фичи до рабочей сборки.

## Важно о текущем статусе проекта

- Основной рабочий рендер: `DirectX12` (Windows).
- `Vulkan` сейчас в виде заглушки.
- Редактор построен на Dear ImGui (при наличии `third_party/imgui-1.90.9`).
- В проекте есть voxel sandbox как референс игрового модуля.
